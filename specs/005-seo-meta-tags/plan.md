# Implementation Plan: SEO & Meta Tags

**Branch**: `005-seo-meta-tags` | **Date**: 2026-02-19 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `specs/005-seo-meta-tags/spec.md`

## Summary

Add SEO and social sharing discoverability to the Queems SPA by editing `index.html` to include a descriptive title, meta description, robots directive, canonical link, Open Graph tags, Twitter Card tags, and a JSON-LD structured data block. A 1200×630 PNG preview image will be created as a static public asset. No new library dependencies are introduced.

## Phase 1: Design

### index.html Changes

The complete set of tags to add to `<head>` (in addition to existing charset/viewport/favicon):

```html
<!-- SEO fundamentals -->
<title>Queems — Queens Puzzle Game</title>
<meta name="description" content="Place queens on a color-coded board so no two queens share a row, column, region, or diagonal. A browser-based N-Queens puzzle game." />
<meta name="robots" content="index, follow" />
<link rel="canonical" href="https://queems.app" />

<!-- Open Graph (Facebook, WhatsApp, LinkedIn) -->
<meta property="og:title" content="Queems — Queens Puzzle Game" />
<meta property="og:description" content="Place queens on a color-coded board so no two queens share a row, column, region, or diagonal." />
<meta property="og:type" content="website" />
<meta property="og:url" content="https://queems.app" />
<meta property="og:image" content="https://queems.app/og-image.png" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
<meta property="og:image:type" content="image/png" />
<meta property="og:image:alt" content="Queems puzzle board with color regions and queens" />
<meta property="og:site_name" content="Queems" />
<meta property="og:locale" content="en_US" />

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="Queems — Queens Puzzle Game" />
<meta name="twitter:description" content="Place queens on a color-coded board so no two queens share a row, column, region, or diagonal." />
<meta name="twitter:image" content="https://queems.app/og-image.png" />
<meta name="twitter:image:alt" content="Queems puzzle board with color regions and queens" />

<!-- Structured Data (JSON-LD) -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebApplication",
  "name": "Queems",
  "description": "Place queens on a color-coded board so no two queens share a row, column, region, or diagonal. A browser-based N-Queens puzzle game.",
  "url": "https://queems.app",
  "applicationCategory": "GameApplication",
  "image": "https://queems.app/og-image.png",
  "creator": {
    "@type": "Person",
    "name": "Ron Hachmon"
  },
  "inLanguage": "en-US",
  "isAccessibleForFree": true,
  "datePublished": "2026-02-01"
}
</script>
```
